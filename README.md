# SplashAssessment

## Getting Started

First, in game-back-end and in game-front-end, after cloning the project please run :
```bash
npm install
```

when all is installed you can run
in game-back-end :
```bash
node app.js
```

in game-front-end :
```bash
npm run dev
```

after this, you can access the game through :
http://localhost:3000/

Thanks.
